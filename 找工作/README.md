# 找工作 AI Agent - 使用指南

## 快速开始

### 1. 填写简历
编辑 `resume.md` 文件，填写你的完整简历信息。

### 2. 配置偏好
编辑 `config.json` 文件，设置：
- 期望职位
- 期望薪资
- 工作地点
- 公司类型偏好
- 其他筛选条件

### 3. 添加职位信息

#### 方式 1：手动添加（推荐）
1. 复制招聘网站上的职位信息
2. 参考 `jobs/job-template.json` 格式
3. 创建新的 JSON 文件保存到 `jobs/zhilian/` 或 `jobs/lagou/` 目录

#### 方式 2：使用爬虫脚本（需要技术基础）
```bash
cd scripts
python3 scraper.py
```
注意：爬虫脚本仅供参考，实际使用需要处理反爬虫、登录等问题。

### 4. 分析职位

#### 使用 Claude AI 分析（推荐）
在 Claude Code 中输入：
```
分析 jobs/zhilian/ 目录下的所有职位
```

#### 使用批量处理脚本
```bash
cd scripts
python3 batch-process.py
```

### 5. 生成沟通信息
```
为匹配度大于 80 分的职位生成打招呼信息
```

### 6. 查看结果
- 分析报告：`analysis/` 目录
- 沟通信息：`messages/` 目录

## 文件说明

### 配置文件
- `CLAUDE.md` - AI Agent 配置文档
- `config.json` - 求职偏好配置
- `resume.md` - 简历（Markdown 格式）
- `resume.json` - 简历（结构化数据）

### 数据目录
- `jobs/` - 职位信息
  - `zhilian/` - 智联招聘职位
  - `lagou/` - 拉钩招聘职位
  - `job-template.json` - 职位信息模板
- `analysis/` - 职位分析报告
- `messages/` - 生成的沟通信息
- `sent/` - 已发送记录

### 脚本工具
- `scripts/scraper.py` - 爬虫脚本（参考）
- `scripts/batch-process.py` - 批量处理脚本

## 工作流程

```
1. 收集职位信息
   ↓
2. AI 分析匹配度
   ↓
3. 生成分析报告
   ↓
4. 生成沟通信息（匹配度 > 70 分）
   ↓
5. 人工审核调整
   ↓
6. 发送给招聘方
   ↓
7. 记录发送状态
```

## 注意事项

### 隐私保护
- 不要将简历文件提交到公共代码仓库
- 不要在沟通信息中包含敏感个人信息
- 定期清理过期的职位数据

### 质量控制
- 每条沟通信息都应人工审核后再发送
- 根据公司文化调整沟通风格
- 避免使用过于模板化的语言

### 效率优化
- 优先处理高匹配度职位（>80 分）
- 批量生成，但个性化调整
- 记录发送时间，避免重复投递
- 每天控制发送数量（建议 10-20 个）

## 常见问题

**Q: 如何提高匹配度？**
A: 
1. 完善简历，突出核心技能和项目经验
2. 调整 config.json 中的筛选条件
3. 针对性学习职位要求中的技能

**Q: 生成的沟通信息可以直接发送吗？**
A: 建议先人工审核和调整，确保信息准确、语言自然、符合目标公司文化。

**Q: 如何获取招聘网站的职位信息？**
A: 
1. 手动复制职位信息（推荐）
2. 使用浏览器插件导出
3. 运行爬虫脚本（需要处理反爬虫）

**Q: 可以同时处理多个招聘平台吗？**
A: 可以，将不同平台的职位信息放在对应的子目录下即可。

## 技巧与建议

### 简历优化
- 使用量化数据展示成果
- 突出与目标职位相关的经验
- 保持简历在 2-3 页
- 定期更新简历内容

### 职位筛选
- 设置合理的期望薪资范围
- 关注公司发展阶段和文化
- 优先选择技术栈匹配的职位
- 避免频繁跳槽的公司

### 沟通技巧
- 开场简洁有力
- 突出与职位最相关的经验
- 表达对公司业务的兴趣
- 保持专业和礼貌

### 跟进管理
- 记录每次投递的时间
- 3-5 天后可以礼貌跟进
- 面试后及时发送感谢信
- 保持积极的心态
